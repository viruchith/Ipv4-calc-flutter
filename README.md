# IPv4 Address Calculator App

A Simple Flutter application to calculate
- Network IP,Broadcast IP
- First usable IP
- Last Usable IP
- Prefix
- Hosts  
   
for a given Ipv4 Address and Subnet Mask.

- URL to IP conversion feature  is also available .
